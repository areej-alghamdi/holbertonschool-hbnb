# HBnB Evolution Project

This repository contains the full-stack development phases of the **HBnB Evolution** application, a property rental platform. The project is managed and implemented progressively across multiple technical layers.

---

## 🗺️ Project Development Phases (A to Z)

The development lifecycle of this project is divided into four distinct phases to ensure system scalability and clean architecture:

* **Phase 1: Technical Documentation & System Architecture**
    * Designing UML Class Diagrams for entity relationships.
    * Constructing System Sequence Diagrams for core workflows.
    * Documenting high-level package architecture and design choices.

---

## 📂 Repository Layout

* `part1/`: Contains all deliverables for the architecture and design phase (UML models, documentation, and flows).
* `README.md`: This file, providing a high-level overview of the global project scope.
